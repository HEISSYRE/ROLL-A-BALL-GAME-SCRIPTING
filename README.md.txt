## Enhancement Feature Implementation

### Enemy AI System

An enemy character has been added as an enhancement feature to extend the core gameplay experience beyond the basic assignment requirements.

The enemy uses Unity’s NavMeshAgent system to dynamically track and follow the player throughout the level. This creates a simple AI-driven chase mechanic that increases difficulty and adds interactivity to the game.

This system serves as the project’s enhancement component and demonstrates the use of Unity pathfinding and AI navigation beyond the required specifications.
